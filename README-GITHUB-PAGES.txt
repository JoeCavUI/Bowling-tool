Upload all files in this folder to the root of your GitHub repository.

Then in GitHub:
- Settings -> Pages
- Source: Deploy from branch
- Branch: main
- Folder: /(root)

Your site will publish at:
https://<your-username>.github.io/<repo-name>/

Do not upload the outer bowling-app folder, node_modules, package.json, or server.js.
Only upload the files in this folder.
