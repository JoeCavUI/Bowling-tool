Bowling Tool v66 clean build

How to use
1. Unzip this folder.
2. Open index.html directly on your laptop.
3. Use Home > Export backup to save your data as JSON.
4. Move that JSON file and this whole folder to your iPhone.
5. Open index.html on the iPhone in Safari or Files, then use Import backup.

Notes
- This build does not require npm or localhost.
- Only the current files are included. Old version scripts were removed.
- Your saved data stays in the browser/device local storage until you clear site data.
